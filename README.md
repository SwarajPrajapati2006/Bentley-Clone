## 🏎️ About This Project

> **Bentley Motors Clone** is a pixel-perfect, responsive frontend clone of the official [Bentley Motors](https://www.bentleymotors.com) website. Built using pure HTML, CSS, and vanilla JavaScript — it replicates the luxury feel, smooth animations, glassmorphism navbar, and elegant layout of one of the world's most prestigious automotive brands.

  <img src="https://user-images.githubusercontent.com/74038190/212284100-561aa473-3905-4a80-b561-0d28506553ee.gif" width="100%" alt="divider">
</div>

## ✨ Features

### 🧭 Navigation
- **Glassmorphism Navbar** — frosted glass effect on page load
- Navbar transitions to a **solid dark background** on scroll
- **Hamburger menu** for mobile responsiveness
- Links to Models, Culture, Lifestyle, Your Bentley, Configurator, Locate Dealer
- **Test Drive** CTA button in the navbar

---

### 🎬 Hero Section
- Full-screen **autoplay background video** (looped, muted)
- Cinematic dark overlay for text contrast

---

### 📖 Intro Section
- Brand story content about Bentley Motors & Beyond100 sustainability manifesto
- **CTA buttons** — About Bentley & Create Your Own
- Smooth **scroll reveal animations** using Intersection Observer API

---

### 🚗 Models Grid
- **3-card grid layout** with luxury car images as backgrounds
- Cards for **Models**, **Pre-Owned Bentley**, and **Factory**
- Staggered **delay animations** on scroll reveal

---

### 🔀 Feature Split Sections
- **Mulliner** section — dark theme, split image + content layout
- **Configure Your Perfect Car** — gold theme, reverse split layout
- Each section has its own CTA button

---

### 🌍 About Bentley Section
- Portrait image + text layout
- Talks about Bentley's sustainability journey & electric future
- Dual CTA — **Explore** & **Keep Me Informed**

---

### 🔗 Footer
- Top footer with Newsletter, Configurator, Locate Dealer, Download Brochure links
- **Bentley logo** centered in footer
- **Social media icons** — Instagram, LinkedIn, YouTube, Twitter, Facebook
- Legal links — Sitemap, Contact, Terms, Privacy, Cookie Policy, Battery Passport, EU Data Act
- Registered office details

---

### 📄 Additional Pages
- **Lifestyle Page** — `/lifestyle/lifestyle.html`
- **Test Drive Page** — `/test drive/drive.html`
- **Activity Page** — `activity.html`

## 🚀 Getting Started
```bash
# Clone the repository
git clone https://github.com/SwarajPrajapati2006/Bentley-Clone.git

# Navigate into the project
cd Bentley-Clone

# Open in browser
open index.html
```

> No dependencies, no build tools — just open `index.html` in any browser!
